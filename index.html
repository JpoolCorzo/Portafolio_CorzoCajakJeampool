1. Temas aprendidos

  • Durante la primera semana se realizó la exposición del sílabo del curso Desarrollo de Aplicaciones Web a cargo del docente Mg. Jaime Suasnabar Terrel. Se explicó la estructura general del curso, los temas que se abordarían durante el semestre y las competencias a desarrollar.
  • El docente brindó una introducción a las tecnologías web, explicando los conceptos de sistema web, aplicación web, sitio web y página web, así como los principales protocolos de comunicación (DNS, TCP/IP, HTTP). También se presentaron los roles en el desarrollo de aplicaciones para Internet, diferenciando entre desarrollador frontend, backend y fullstack.
Finalmente, se introdujo la herramienta Visual Studio Code (VSC), detallando sus elementos de interfaz, extensiones y atajos de teclado.

2. Ejercicios de laboratorio - Resultados

  • Se desarrolló el Laboratorio 01A, donde se exploró el entorno de trabajo de Visual Studio Code, configurando extensiones útiles para el desarrollo web y familiarizándose con su uso práctico.

3. Reflexión

  • La primera semana permitió comprender la organización del curso, los objetivos de aprendizaje y las herramientas a utilizar. Además, se afianzaron los conocimientos básicos sobre la web y su funcionamiento, sentando las bases para el desarrollo de aplicaciones web frontend y backend durante el semestre.
